# DragonSea MC Web

A sleek, modern website for the DragonSea Minecraft Server, featuring a dark theme with glassmorphism effects, smooth animations, and a contemporary design.

## Pages

- **index.html**: Home page with server links, Dynmap, transit system, and server status
- **about.html**: About page with community information and features

## Features

- **Navigation Bar**: Fixed top navigation with Home and About links
- **Server Status Card**: Real-time server status display (currently mocked)
- **Server Link**: Direct connection to the Minecraft server
- **Dynmap**: Real-time world map viewer
- **Transit System**: Guide for the server's transportation network
- **Internationalization**: Automatic language detection (Chinese for Chinese clients, English otherwise)
- **Language Switcher**: Button in the top right corner to toggle between English and Chinese
- **Keyboard Shortcuts**: Press 'E' for English, 'C' for Chinese to manually switch languages

## Technologies

- HTML5
- CSS3 (with modern gradients, glassmorphism, animations, and Inter font)
- JavaScript (for interactivity and language switching)

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Or serve it with a local server for better experience

## Customization

Update the links in `index.html` to point to your actual server URLs.

Colors can be adjusted in `css/style.css`.
